# 介绍
 - 根据我们的业务抽离出常见的一些图表，封装成组件。基于 Vue 和 echarts 封装的 x-echarts 图表组件，只需要统一提供一种对前后端都友好的数据格式设置简单的配置项，便可轻松生成常见的图表。
 - 我们做业务的时候，可以参考看下，如果没有找到对应的配置，也可以看下echarts官方文档。
