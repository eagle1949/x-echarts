# 颜色说明

#### 白底配色
图表默认使用配色如下：

	 '#5470C6', '#91CC75', '#FACB59', '#EE6666', '#73C0DE', '#3BA272', '#FD8C56', '#9A60B4', '#EA7CCC'

#### 深色底配色
图表默认使用配色如下：

	 '#4B97FF', '#7FFFB7', '#FDDD60', '#FF6E76', '#58D9F9', '#05C091', '#FF8A45', '#8D48E3', '#DD79FF'
    
#### 渐变色示例
`{
	x: 0,
	y: 0,
	x2: 0,
	y2: 1,
	type: 'linear',
	global: false,
	colorStops: [
		{
			offset: 0,
			color: 'rgba(10,219,250,0.5)'
		},
		{
			offset: 1,
			color: 'rgba(10,219,250, 0)'
		}
	]
}`
